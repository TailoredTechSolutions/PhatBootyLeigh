## What This Is

A fully custom, single-file event management dashboard built for **Host Leigh** — Metro Manila's award-winning corporate event host, emcee, and production coordinator.

Designed to run from a single HTML file. No framework, no build step, no internet required.

---

## Features

| Module | What it does |
|--------|-------------|
| 🗓 Command Center | Live countdown · KPI cards · SideRays WebGL hero · Revenue chart |
| 💬 Supplier War Room | Auto-reads program flow → one-tap WhatsApp per supplier |
| 📋 Gig Pipeline | Drag-and-drop Kanban across 5 stages |
| 🗂 Client CRM | Tier system · payment ratings · event history |
| 🔧 Supplier Firewall | Tier ranking · tawad simulator · suki milestone tracker |
| 💰 Escrow Tracker | Per-event inbound/outbound · margin calculator |
| 👗 Wardrobe Planner | 8 outfits · assign to events · avoid repeats |
| 📄 Program Flow Builder | Drag-and-drop cue blocks · send to WhatsApp |
| 📅 Calendar | iOS-style dark luxury grid · iCloud CalDAV sync |
| 👥 Client Portal | Client-facing event timeline and payment progress |

---

## Visual Effects

- **SideRays** — Raw WebGL hero background, gold `#d4af37` + rose `#b4354e` rays (ported from React Bits, no OGL dependency)
- **BorderGlow** — CSS custom-property hover glow on KPI cards, per-card accent colours (ported from React Bits, pure CSS + vanilla JS)
